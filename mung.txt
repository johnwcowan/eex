Options:
-M      modify no files (can't write any file)
-R      read-only (can't write input files)
-A      autoread changed files
-W      autowrite files when moving away
-B      back up files on write

Commands:
|               filter lines through command (\ for multiline)
args            show arguments
cd              change directory
choices         list redo choices
exit            write all modified files and exit
file            file metadata
flush           discard current file
mung            append new file(s)
next            move to next file
preserve        save complete state
prev            move to previous file
quit            flush all files and exit
read            insert file into buffer
redo            undo undo (option n to choose one)
rename          change name of current file
rewind          jump back to first buffer
set             set option
shell / !       run shell
suspend / stop  suspend process
undo            undo last change
version         display version and other info
write           write buffer (nosam)
???             command to output pipeline
